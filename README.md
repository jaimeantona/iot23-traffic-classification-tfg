# Detección de anomalías en redes IoT mediante la combinación de técnicas estadísticas y de aprendizaje automático

Código desarrollado para el Trabajo Fin de Grado de Jaime Antona Díaz, realizado en la Escuela Politécnica Superior de la Universidad Autónoma de Madrid.

El repositorio contiene los *scripts* utilizados para la construcción de los conjuntos de datos a partir de *IoT-23*, el entrenamiento de los modelos de clasificación y la generación de las figuras empleadas en la memoria.

- data_processing/   Procesamiento y construcción de los conjuntos de datos
- training/          Entrenamiento y evaluación de modelos
- figures/           Generación de figuras de resultados


Las capturas originales y los conjuntos de datos procesados no se incluyen en el repositorio debido a su tamaño.
